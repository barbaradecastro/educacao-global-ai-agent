# Education Insight Skill

## Objetivo

A **Education Insight Skill** foi desenvolvida para apoiar a análise de indicadores educacionais utilizando dados do **World Bank Education Statistics (EdStats)**.

Seu objetivo é transformar dados processados pelo pipeline em informações estratégicas, permitindo comparar países, identificar tendências históricas e gerar análises executivas para apoio à tomada de decisão.

---

# Capacidades

A Skill é capaz de:

- selecionar países para análise;
- selecionar indicadores educacionais;
- comparar o desempenho entre países;
- gerar rankings dos indicadores analisados;
- produzir insights em linguagem natural;
- gerar relatórios executivos baseados nos dados.

---

# Entradas Esperadas

A Skill recebe como entrada:

- lista de países;
- lista de indicadores;
- período de análise;
- dados processados pelo pipeline.

---

# Saídas Produzidas

A Skill produz:

- resumo executivo;
- comparação entre países;
- ranking dos indicadores;
- principais insights;
- recomendações;
- limitações da análise.

---

# Casos de Uso

A Skill pode ser utilizada em:

- monitoramento de indicadores educacionais;
- apoio à tomada de decisão;
- pesquisas acadêmicas;
- geração automática de relatórios;
- agentes inteligentes baseados em LLMs.

---

# Integração

A Skill foi projetada para funcionar em conjunto com o pipeline principal do projeto **Global Education AI Agent**, utilizando os dados processados e enriquecidos durante a execução da solução.
