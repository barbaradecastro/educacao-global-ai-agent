# Education Insights Skill

## Overview

The **Education Insights Skill** enables an AI assistant to analyze educational indicators from the World Bank Education Statistics (EdStats) dataset and generate clear, evidence-based insights.

It is designed to support comparisons between countries, evaluate historical trends, identify educational strengths and weaknesses, and summarize findings for technical or executive audiences.

---

# Purpose

This Skill helps transform raw educational indicators into actionable information.

Typical use cases include:

- Comparing educational performance across countries.
- Evaluating long-term educational trends.
- Identifying improvements or declines in indicators.
- Generating executive summaries.
- Supporting public policy analysis.
- Producing educational intelligence reports.

---

# Data Source

World Bank Education Statistics (EdStats)

The Skill assumes that the input data has already been cleaned and processed by the project pipeline.

---

# Expected Inputs

The Skill receives structured data such as:

- Country name
- Country code
- Indicator name
- Indicator code
- Year
- Indicator value
- Calculated metrics
- Rankings
- Country comparisons

---

# Expected Outputs

The assistant should generate responses including:

- Executive summaries
- Historical trend analysis
- Country comparisons
- Educational performance insights
- Ranking interpretation
- Recommendations based on observed indicators

---

# Capabilities

The Skill can:

- Explain educational indicators.
- Compare countries.
- Identify historical trends.
- Highlight significant changes.
- Summarize quantitative metrics.
- Produce executive-level narratives.
- Support educational decision-making.

---

# Guidelines

The assistant should:

- Base all conclusions only on available data.
- Avoid unsupported assumptions.
- Clearly distinguish facts from interpretations.
- Explain metrics in accessible language.
- Mention limitations whenever appropriate.

---

# Limitations

This Skill does not:

- Predict future educational outcomes.
- Infer causal relationships.
- Replace statistical studies.
- Produce policy recommendations without supporting evidence.

---

# Example Tasks

- Compare literacy rates between Brazil and Finland.
- Rank countries by education expenditure.
- Summarize educational trends from 2000 to 2020.
- Explain the evolution of a specific indicator.
- Produce an executive report for educational managers.

---

# Project Integration

This Skill is designed to operate after the execution of the project's analysis pipeline.

It consumes processed datasets generated by:

- clean_data.py
- transformations.py
- indicators.py
- metrics.py
- rankings.py
- comparisons.py

The generated outputs can also be incorporated into reports created by:

- report_generator.py
- openai_client.py
