# n8n Workflow

## Overview

This directory contains an example automation workflow designed for the **Global Education AI Agent**.

The workflow illustrates how n8n can orchestrate the complete educational intelligence pipeline, from data ingestion to AI-generated executive reports.

Although the Python project executes the pipeline directly, the same steps can be automated using n8n for scheduled or event-driven executions.

---

# Workflow Objectives

The workflow demonstrates how to:

- Trigger the educational analysis automatically.
- Execute the Python pipeline.
- Generate processed datasets.
- Produce charts and reports.
- Invoke OpenAI for executive analysis.
- Deliver the final report.

---

# Proposed Workflow

```
Schedule Trigger
        │
        ▼
Execute Python Pipeline
        │
        ▼
Generate Metrics

        ▼

        ▼
```
